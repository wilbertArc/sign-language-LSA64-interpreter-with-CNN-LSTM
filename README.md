WELCOME TO OUR SIGN LANGUAGE INTERPRETER
======================================================
This is our sign language interpreter.

To use this, you can:
1. Clone this repo
2. Make sure you have an virtual environment for this
3. Run this command in your terminal:
   python -m streamlit run app.py

FYI: to get the best result, you should wear red gloves on your right hand and green gloves on your left, as this model trains on dataset that utilizes on the colors of the gloves
